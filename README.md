# Secure Password Generator

## Description
This project is a simple CLI (command line) tool that generates secure random passwords. The user can customize password length and choose which characters to include.

## Why this is important
Weak passwords are one of the main reasons accounts get hacked. This tool helps users generate strong and unpredictable passwords.

## Features
- Generate random passwords
- Choose password length
- Option to include/exclude:
  - letters
  - numbers
  - symbols
- Uses secure randomness (`secrets` module)

## How to run

Run the program:

```bash
python3 password_generator.py
